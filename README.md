# Netflix Titles Dashboard – Power BI Project

The Netflix Titles Dashboard is an interactive data visualization project built using Microsoft Power BI.
It provides insights into Netflix's content library including movies and TV shows

## Project objective 
To analyze and visualize Netflix’s content data to uncover trends in distribution, growth, genres, ratings, and country contributions through an interactive Power BI dashboard.

## Dataset used 
<a href="https://github.com/prachi-lp/Data-Analytics--Netflix-Dashboard/blob/main/Netflix%20Titles%20Dashboard.xlsx"> Dataset</a>

## Questions (KPIs)
- What is the total number of titles available on Netflix?
- What is the total duration (in minutes) of all content?
- What is the minimum (oldest) release year in the dataset?
- What is the average release year of all titles?
- What percentage of total content is Movies vs TV Shows?
- How many titles were added in the most recent year?
- Which rating category has the highest number of titles?
- What is the total number of titles added after 2008?
- Which country contributes the highest number of titles based on KPI metrics?
- What is the year with the maximum content addition?

## Process
- Imported the Netflix dataset into Power BI and reviewed its structure.
- Cleaned and transformed the data using Power Query by removing nulls and correcting formats.
- Created DAX measures and built an optimized data model.
- Designed an interactive dashboard with KPIs and visuals to generate meaningful insights.

## Project Insights
- Movies dominate the platform, making up around 67% of the total content.
- The United States contributes the highest number of titles, followed by India and the United Kingdom.
- Netflix experienced rapid content growth after 2008, reflecting global expansion.
- Drama and International genres have the strongest representation in the catalog.
- The platform includes a wide range of content, from classic titles (1925) to recent releases.

## Conclusion 
The Netflix Titles Dashboard successfully transforms raw data into meaningful and interactive business insights using Power BI. The project highlights content distribution, growth trends, country contributions, and genre popularity through clear KPIs and visualizations. It demonstrates strong skills in data cleaning, modeling, DAX calculations, and dashboard design. Overall, the dashboard provides a comprehensive understanding of Netflix’s content strategy and growth patterns in a visually engaging format.
